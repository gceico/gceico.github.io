**AIHAR** turns Claude Code plugins hosted on GitHub into productized **MCP endpoints**. Creators share private harnesses, prompts, skills, and workflows, with customers through signed URLs, so end users get results without ever seeing the underlying code. Every session runs in a sandboxed, isolated environment, with a free cloud tier and a self-hostable open-core engine. Live at [aihar.app](https://aihar.app).
